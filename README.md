# ImcApp

Este projeto foi gerado com [Angular CLI](https://github.com/angular/angular-cli) versão 16.2.1.

## Instalando pacotes

```
npm instal
```

## Rodando em modo de desenvolvimento

```
npm run start
```

## Build

```
npm run build
```

Os artefatos gerados serão armazenados no diretório `dist/`.
